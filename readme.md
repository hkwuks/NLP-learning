## 二、评测标准

评价标准为类别f1_score的均值，选手提交结果与实际测试集的类别进行对比，结果越大越好。

计算公式：$F1 = 2 * \frac{(precision * recall) }{(precision + recall)} $​	

可以通过sklearn完成f1_score计算：

```python
from sklearn.metrics import f1_score
y_true = [0, 1, 2, 0, 1, 2]
y_pred = [0, 2, 1, 0, 0, 1]
f1_score(y_true, y_pred, average='macro')
```
